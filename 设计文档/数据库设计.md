# 数据库设计

## 用户(customer)：
数据名|数据类型|说明
--|:--|:--
id |longint|用户的数据库中的id
userid|varchar|用户的微信id
username|varchar|用户的微信名
logintimes|int|登录次数
level|int|用户的会员等级
credit|int|用户的消费积分
createtime|timestamp|创建时间
updatetime|timestamp|修改时间

## 菜单（menu）
数据名|数据类型|说明
--|:--|:--
id|longint|数据库中的id
name|varchar|菜品名字
menulevel|int|菜单等级，第几级菜单
menunumber|int|菜单编号，属于哪个类
type|int|菜品类型
tasteid|int|口味信息id
